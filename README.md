# AI Project OS

Универсальная операционная система для управления AI-проектами.

## Основные возможности

- **Автоопределение проекта (Detector)**: Автоматическое распознавание типа проекта (MkDocs, Astro, Next.js, Flask, Django, Telegram Bot) и используемого стека технологий.
- **Универсальные настройки (Bootstrapper)**: Автоматическая инициализация `project.json` и `settings.json` на основе обнаруженного стека.
- **Интеграции**: Поддержка Git, Telegram уведомлений и экспорта отчетов в PDF.
- **Модульность**: Гибкая архитектура, позволяющая легко добавлять поддержку новых типов проектов.

## Использование

Для автоматической настройки проекта в новом репозитории выполните:

```powershell
python scripts/core/bootstrapper.py
```

Система проанализирует файлы в корневой директории и создаст необходимую конфигурацию в папке `data/`.
