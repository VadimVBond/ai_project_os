# AI Project OS Scripts

This directory contains utility scripts for managing the project.

## Available Scripts

### `generate_report.ps1`
Generates a Markdown report based on templates and `project.json`.

**Usage:**
```powershell
./scripts/generate_report.ps1 -Type "client"
./scripts/generate_report.ps1 -Type "developer"
```

### `update_progress.ps1` (Planned)
Helper script to update module progress in `project.json` via CLI.
