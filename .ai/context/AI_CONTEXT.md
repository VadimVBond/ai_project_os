## Purpose

AI Project OS is a universal framework for managing software projects with AI assistance.

The framework is designed to work with any technology stack:

* Python
* Next.js
* React
* Vue
* Astro
* MkDocs
* Telegram Bot
* WebApp
* FastAPI
* Django
* SaaS
* Mobile Apps
* Documentation Projects

---

## Main Goal

Provide:

* project tracking
* progress management
* AI collaboration
* customer reporting
* developer reporting

without dependency on project technology.

## Core Principles

Progress is measured through:

* modules
* milestones
* complexity
* completion
