---

## 1. STATE SNAPSHOT (ФАКТЫ)

- текущая версия

- список модулей (name + status + progress)
- последние 5 изменений из changelog

❗ Никаких рекомендаций на этом этапе

---

## 2. CONSISTENCY CHECK

Проверь:

- есть ли расхождения между:
  - project.json
  - PROJECT_STATE.md
  - changelog

- есть ли:
  - завышенный прогресс
  - несоответствие статусов
  - “дубликаты завершения”

Выведи:
- OK / DRIFT DETECTED

Если DRIFT → перечисли проблемы
